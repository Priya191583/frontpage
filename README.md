# ContentGuard
Intelligent Content Similarity &amp; Plagiarism Detection System
ContentGuard is a robust, multi-level plagiarism detection system designed to identify exact, partial, and structurally altered plagiarism in both source code and text documents. By combining advanced string-matching algorithms with dynamic programming and greedy optimization, it moves beyond simple character comparison to detect "structural plagiarism" where logic is replicated even if the syntax is altered.
